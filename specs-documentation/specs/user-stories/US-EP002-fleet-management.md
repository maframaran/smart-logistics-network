# User Stories — EP-002 Fleet Management

## Carrier Stories

**US-012** As a Carrier, I want to register my vehicles on the platform with their type, weight/volume capacity, and special capabilities so that they can be considered for shipment assignments.
_Priority: Must | Feature: F-007_

**US-013** As a Carrier, I want to mark a vehicle as under maintenance so that it is not assigned shipments while unavailable.
_Priority: Must | Feature: F-008_

**US-014** As a Carrier, I want to take a vehicle out of service permanently so that it is removed from the assignment pool.
_Priority: Should | Feature: F-008_

**US-015** As a Carrier, I want to see which of my vehicles are currently available, assigned, or under maintenance so that I can manage my fleet efficiently.
_Priority: Must | Feature: F-008_

## Platform Administrator Stories

**US-016** As a Platform Administrator, I want to query which vehicles can accommodate a specific cargo spec so that I can validate assignment feasibility manually.
_Priority: Should | Feature: F-009_

**US-017** As a Platform Administrator, I want to reinstate a decommissioned vehicle back to available status so that recovered assets can re-enter the fleet.
_Priority: Could | Feature: F-008_
