# User Stories — EP-004 Route Optimization

## Shipper Stories

**US-025** As a Shipper, I want to see the estimated delivery time and cost when my shipment is created so that I can evaluate whether to proceed.
_Priority: Must | Feature: F-013_

## Carrier Stories

**US-026** As a Carrier, I want my drivers to receive an optimized route that minimizes fuel costs so that my operational expenses are reduced.
_Priority: Must | Feature: F-014_

## Platform Administrator Stories

**US-027** As a Platform Administrator, I want to configure route optimization weights (time vs. cost vs. tolls) per SLA tier so that Express shipments prioritize speed and Standard shipments prioritize cost.
_Priority: Should | Feature: F-014_

**US-028** As a Platform Administrator, I want to be alerted when route calculation fails for a shipment so that I can intervene and prevent SLA breaches.
_Priority: Must | Feature: F-013_
