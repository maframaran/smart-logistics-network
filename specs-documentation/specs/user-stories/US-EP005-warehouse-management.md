# User Stories — EP-005 Warehouse Management

## Warehouse Operator Stories

**US-029** As a Warehouse Operator, I want to record inbound inventory so that the platform knows what goods are stored and can plan outbound shipments.
_Priority: Must | Feature: F-015_

**US-030** As a Warehouse Operator, I want to be warned before recording inbound inventory that would exceed warehouse capacity so that I can redirect goods to another facility.
_Priority: Must | Feature: F-015_

**US-031** As a Warehouse Operator, I want to prepare and confirm outbound inventory for an assigned shipment so that the driver can pick it up without delay.
_Priority: Must | Feature: F-016_

**US-032** As a Warehouse Operator, I want to see real-time warehouse utilization so that I can plan inbound scheduling and avoid overflow.
_Priority: Must | Feature: F-017_

## Platform Administrator Stories

**US-033** As a Platform Administrator, I want to configure warehouse capacity and operating hours so that the platform enforces correct constraints for that facility.
_Priority: Must | Feature: F-017_

**US-034** As a Platform Administrator, I want to receive an alert when any warehouse reaches 90% capacity so that I can take proactive action before overflow.
_Priority: Should | Feature: F-017_
